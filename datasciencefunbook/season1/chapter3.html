🔍 Let’s Go Deeper…

We all love “insights” — but how confident are we in them? A confidence interval says: "We’re 95% sure it’s between X and Y."

📚 What is a Confidence Interval?

  • A range of values likely to contain the true parameter.
  • Components: Point estimate, Standard Error, Confidence Level.
  • Example: “Average = 3.2 ± 0.6 (95% CI)”

💻 Hands-on – Simulating Confidence Intervals in Python

import numpy as np
import scipy.stats as stats

data = np.random.normal(loc=3.2, scale=0.6, size=100)

mean = np.mean(data)
sem = stats.sem(data)

confidence = 0.95
interval = stats.t.interval(confidence, df=len(data)-1, loc=mean, scale=sem)

print(f"Sample mean: {mean:.2f}")
print(f"95% confidence interval: {interval}")

✅ Lessons Learned

  • Confidence intervals quantify uncertainty in estimates.
  • They’re key for decisions, tests, and reporting.
  • Wider = more uncertainty; narrower = more data/less variance.

⚠️ Common Mistakes

  • Confusing confidence vs. prediction intervals.
  • Misinterpreting the 95% CI statement.
  • Ignoring sample size — small samples = big error margins.
